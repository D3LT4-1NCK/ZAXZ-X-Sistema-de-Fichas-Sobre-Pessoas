# Sistema de Fichas Sobre Pessoas
_Neste sistema escrito em Python + SQL, você pode anotar de forma organizada, várias informações específicas sobre pessoas; no entanto, não anote informações de pessoas sem autorização. Só testado em Linux._

<big><big><big>🖥️ <b>Detalhes:</b></big></big></big>

**Versão:** _1.0_ <br>

<big><big><big>🔨️ <b>Desenvolvimento:</b></big></big></big>

- _Projeto simples que considero importante para quem não quer ter centenas de anotações separadas em múltiplos lugares._
- _Esse projeto não descobre informações sobre pessoas, e nem usa qualquer tipo de ferramenta hacker._
- _Ainda irei criar a possibilidade de apagar pessoas da lista._
- _E também estou a procura de uma solução para o erro que,quando a tabela passa da área do Terminal, acaba virando algo totalme ilegível... Talvez nunca seja arrumado, realmente não achei uma opção para isso sem trabalhar com front end._

<big><big><big>⚠️ <b>Responsabilidades do Usuário:</b></big></big></big>

- _A responsabilidade é totalmente sua ao registrar dados de terceiros sem o consentimento deles. Haja com consciência e respeite a privacidade alheia._
