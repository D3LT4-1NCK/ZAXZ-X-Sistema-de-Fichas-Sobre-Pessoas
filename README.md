# Sistema de Fichas Sobre Pessoas
_Neste sistema escrito em Python + SQL, você pode anotar de forma organizada, várias informações específicas sobre pessoas; no entanto, não anote informações de pessoas sem autorização. Só testado em Linux._

<big><big><big>🖥️ <b>Detalhes:</b></big></big></big>

**Versão:** _1.2_ <br>

<big><big><big>🔨️ <b>Desenvolvimento:</b></big></big></big>

- _Projeto simples que considero importante para quem não quer ter centenas de anotações separadas em múltiplos lugares._
- _Esse projeto não descobre informações sobre pessoas, e nem usa qualquer tipo de ferramenta hacker._
- _Ainda irei criar a possibilidade de apagar pessoas da lista._
- _E também estou a procura de uma solução para o erro que,quando a tabela passa da área do Terminal, acaba virando algo totalme ilegível... Talvez nunca seja arrumado, realmente não achei uma opção para isso sem trabalhar com front end._

<big><big><big>⚠️ <b>Responsabilidades do Usuário:</b></big></big></big>

- _A responsabilidade é totalmente sua ao registrar dados de terceiros sem o consentimento deles. Haja com consciência e respeite a privacidade alheia._

<big><big><big>📦 <b>Requisitos:</b></big></big></big>

- _SQLite 3._
- _Python 3+._
- _Git._

<big><big><big>🔧 <b>Como Instalar:</b></big></big></big>

_- 1- Procure como instalar o SQLite 3 na sua distro._ <br>
_- 2- Caso não tenha instalado Python 3 e Git, faça o mesmo._ <br>
_- 3- Agora, no terminal, digite: `git clone https://github.com/D3LT4-1NCK/Sistema-de-Fichas-Sobre-Pessoas ~/ZAXZ_X_TEMP && mv ~/ZAXZ_X_TEMP/ZAXZ\ X ~/ && rm -rf ~/ZAXZ_X_TEMP`_ <br>

_- 4- Agora, se você usa BASH, digite: `nano ~/.bashrc`_ <br>
_- 4- Já se você usa ZSH, digite: `nano ~/.zshrc`_ <br>

_- 5- Desça o máximo que puder e coloque:_ <br>
`zaxz_x() {` <br>
`   python3 ~/ZAXZ\ X/x_main.py` <br>
`}` <br>

_- 6- Aperte 'CTRL' + 'X', depois 'S' ou 'Y' e por fim 'Enter'._ <br>
_- 7- Novamente, se você usa BASH, digite: `source ~/.bashrc`_ <br>
_- 7- No entanto, se você usa ZSH, digite: `source ~/.zshrc`_ <br>
_- 8- E pronto! Basta digitar: `zaxz_x`_ <br>
_- Atenção: Caso o passo 8 não tenha funcionado, feche e abra o terminal, e digite `zaxz_x`_ <br>


<big><big><big>🪛️ <b>Como Atualizar:</b></big></big></big>

_- Basta abrir o terminal e digitar (Você não vai perder as anotações antigas): `mkdir -p ~/CSV_X_TEMP && cp ~/ZAXZ\ X/infos.csv ~/CSV_X_TEMP/ && rm -rf ~/ZAXZ\ X && git clone https://github.com/D3LT4-1NCK/Sistema-de-Fichas-Sobre-Pessoas ~/ZAXZ_X_TEMP && mv ~/ZAXZ_X_TEMP/ZAXZ\ X ~/ && rm -rf ~/ZAXZ_X_TEMP && rm -f ~/ZAXZ\ X/infos.csv && mv ~/CSV_X_TEMP/infos.csv ~/ZAXZ\ X/ && rm -rf ~/CSV_X_TEMP`_
